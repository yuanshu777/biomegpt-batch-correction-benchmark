# MMUPHin Bridge Notes

The main benchmark is the 389-sample overlap subset identified by public
accession matching between MMUPHin CRC metadata and previous BiomeGPT metadata.

Use the full 551-sample MMUPHin CRC dataset only as a reference for raw
abundance versus MMUPHin-adjusted abundance. Do not compare full-551 MMUPHin
metrics directly against 389-sample BiomeGPT metrics as the primary result.

The bridge intentionally keeps accession-derived matches visible in the
manifest. If a match is ambiguous, keep it as possible rather than confirmed.
